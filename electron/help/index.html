About SiteScop V6

SiteScop V6 — Local Edition is a desktop Building & Pest inspection platform built for Australian inspectors. Agreements, jobs, inspection forms, PDF reports, invoices, accounting, and client records are stored on your computer — not in a public cloud.

SiteScop helps you manage the full inspection workflow: send agreements for e-signature, schedule jobs, complete building and pest inspections on site (including voice dictation), generate professional PDF reports, track payments, and email clients — all from one desktop app.

How to open SiteScop

  • Installed app: Desktop shortcut or Start Menu → SiteScop V6.
  • Developer folder: Double-click START-SITESCOP.bat in the sitescop-v6 folder. Keep the black launcher window open while the app runs.
  • Always use the desktop window titled SiteScop V6 — not Chrome, Edge, or another browser tab.

Privacy & your data

  • Inspection data, clients, agreements, and reports live in a local database on this device.
  • Email is sent only when you (or automatic reminders) trigger it via your SMTP settings.
  • GitHub Cloud Signing uploads agreement files only when you enable and use that feature.
  • “Remember me” on login stores your email and password on this PC only.

What’s included in V6

  • Dashboard with today’s jobs and quick stats
  • Agreements with online signing (GitHub Pages optional)
  • Full inspection workspace — building, pest, and combined
  • PDF report generation with your logo and branding
  • Accounting with payment tracking, insights, and Xero sync
  • Calendar with drag-and-drop rescheduling
  • Recycle Bin for deleted jobs and agreements
  • Settings for company, billing, email, reminders, and more

After updates

  • Close every SiteScop window (and the launcher if using the .bat file).
  • Start again so new menus, Help, and features load correctly.
  • If a feature says “Restart app to enable”, run REPAIR-SITESCOP.bat then start again.
  • For a fresh installed copy, run the latest SiteScop V6 Setup 6.0.0.exe installer.
